# skudkey

Программа следит за Telegram-чатом и автоматически регистрирует ключи для домофонов
в СКУД Уфанет.

Когда к домофону прикладывают незарегистрированный ключ, бот `UfanetMQTTWatcher`
пишет в чат сообщение `AC-AUTH-FAIL`. Программа читает это сообщение и сама
добавляет ключ в СКУД. В чат ничего не пишет — только читает.

## Запуск

Нужен Go 1.21+.

```bash
go build -o skudkey .
./skudkey
```

При запуске открывается веб-интерфейс в браузере - там настройки, статус и
список ключей.

## Настройка

Скопируйте `.env.example` в `.env` и заполните значения. Либо задайте настройки
прямо в веб-интерфейсе.

Главное, что нужно указать:

- `MODE` - `history` по умолчанию.
- `SKUD_UNION_ID` - id из ссылки создания ключа в СКУД. Можно найти в адресной строке на странице домофона.
- `SKUD_CONTRACT` и `SKUD_PASSWORD` - логин и пароль от СКУД (программа сама
  получает и обновляет токен).
- `KEY_NAME` - имя, под которым регистрируются новые ключи.
- `SKUD_MAC` - MAC-адрес домофона
